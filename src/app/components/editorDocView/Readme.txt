Pre-Requise:
This page is only visible if the user have a role of "Manager", otherwise it will be move to the main Dashboard or Login page. 

Info:
This page will get the DocID Template from CouchDB and display the info. 

Action: 
The manager can Modify how this doc will behave by modifying the JSON that build that Doc. 
ex: 
- Section 2 before Section 1. 
- Allowing some item to be editable by the user. 
- Display X and not Y. 
- Adjusting the translation.
With 1 click, the manager can see what the final result to the user will look like (Preview function).
